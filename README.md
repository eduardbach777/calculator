# Rock Paper Scissors Game

Ein einfaches Schere-Stein-Papier Spiel gegen den Computer, das ich als Übungsprojekt gebaut habe.

## Was macht das Projekt?

Ein Browser-basiertes Spiel mit:
- Spieler gegen Computer
- Punkte-Tracking (erster zu 5 gewinnt)
- Anzeige von Gewinner/Verlierer
- Reset-Funktion für neues Spiel

## Wie starte ich es?

Einfach `index.html` im Browser öffnen.

Oder mit VS Code Live Server: Rechtsklick auf `index.html` → "Open with Live Server"

## Technisches

- HTML für die Struktur
- CSS für das Design
- Vanilla JavaScript für die Logik
- Keine externen Libraries

## Ordnerstruktur
```
├── index.html
├── style.css
├── script.js
└── README.md
```

## Was ich dabei gelernt habe

- DOM Manipulation und Event Listeners
- If/Else Logik für die Gewinn-Bedingungen
- Math.random() für Zufallszahlen
- CSS Flexbox für das Layout

Das schwierigste war, alle möglichen Gewinn-Kombinationen richtig zu programmieren (wer schlägt wen).
